## 🇪🇸 Español

### 📝 Descripción
Plataforma completa de predicciones (Quiniela) para la Copa del Mundo FIFA 2026. Permite a los usuarios registrarse, realizar predicciones para la fase de grupos y eliminatorias, ver estadísticas en tiempo real y competir en una tabla de posiciones global.

### ✨ Características Principales
- **Sistema de Predicciones:** Interfaz intuitiva para pronosticar resultados de todos los partidos.
- **Fase de Eliminatorias Dinámica:** Algoritmo avanzado que calcula los emparejamientos de octavos, cuartos, semis y final basado en las reglas de la FIFA.
- **Tablero en Tiempo Real:** Visualización de resultados en vivo, estadísticas de equipos y tabla de posiciones.
- **Panel de Administración:** Gestión de usuarios, validación de pagos, actualización de resultados y control de fases del torneo.
- **Sistema de Notificaciones:** Correos electrónicos automáticos para bienvenida, confirmación de pagos y recordatorios de cierre de fase mediante Resend.
- **Seguridad Avanzada:** Autenticación robusta y políticas RLS (Row Level Security) en Supabase.

### 🛠️ Tecnologías
- **Frontend:** Astro 6, React 19, Tailwind CSS.
- **Backend/Base de Datos:** Supabase (PostgreSQL, Auth, Edge Functions).
- **Correos:** Resend con React Email.
- **Despliegue:** Netlify.

### 🚀 Instalación Local
1. Clonar el repositorio: `git clone https://github.com/tu-usuario/quiniela-2026.git`
2. Instalar dependencias: `pnpm install`
3. Configurar variables de entorno: Copia `.env.example` a `.env` y completa tus credenciales de Supabase y Resend.
4. Iniciar servidor de desarrollo: `pnpm dev`

<a name="english"></a>
## 🇺🇸 English

### 📝 Description
A comprehensive prediction platform (Quiniela) for the FIFA World Cup 2026. Users can register, predict group and knockout stage results, view real-time statistics, and compete in a global leaderboard.

### ✨ Key Features
- **Prediction System:** Intuitive interface to forecast results for all matches.
- **Dynamic Knockout Stage:** Advanced algorithm that calculates pairings for round of 32, 16, quarters, semis, and final based on FIFA rules.
- **Real-time Dashboard:** Live scores, team statistics, and standings visualization.
- **Admin Panel:** User management, payment validation, match results updates, and tournament phase control.
- **Notification System:** Automatic emails for welcome, payment confirmation, and phase reminders using Resend.
- **Advanced Security:** Robust authentication and Supabase RLS (Row Level Security) policies.

### 🛠️ Tech Stack
- **Frontend:** Astro 6, React 19, Tailwind CSS.
- **Backend/Database:** Supabase (PostgreSQL, Auth, Edge Functions).
- **Emails:** Resend with React Email.
- **Deployment:** Netlify.

### 🚀 Local Setup
1. Clone the repository: `git clone https://github.com/your-username/quiniela-2026.git`
2. Install dependencies: `pnpm install`
3. Configure environment variables: Copy `.env.example` to `.env` and fill in your Supabase and Resend credentials.
4. Start development server: `pnpm dev`
